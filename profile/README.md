# Thesis 

This GitHub organization is to house my master's thesis codebase and results, as well as attempts and future work.

Readme and repos are WIP

##  Setup
- Ubuntu Linux 22.04
- Python 3.10.12
- Mujoco 3.3.5

Repositories:
- Passive Dynamic Compass Biped - `CompassBiped`
- Finite State Machine (FSM) Walkers - `Planar-Bipeds-FSM`
- SIMBICON - `SIMBICON-Benchmark`
- TALOS LIPM - `Talos-LIPM-ZMP`


## Running Models
1. `python3 -m mujoco.viewer`
2. Drag and drop the `.xml` into the loaded viewer



## Running Scripts
1. `python3 <script name>` - exact commands found in respective repositories.
